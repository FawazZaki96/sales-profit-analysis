# Power BI Dashboard

This folder contains the Power BI file used to build the Sales & Profit Performance dashboard.

## 📊 File

* `superstore_dashboard.pbix`

## 🧾 Description

The dashboard was developed using Power BI to analyze sales and profitability across different categories, regions, and time periods.

It includes:

* Key performance indicators (KPIs):

  * Total Sales
  * Total Profit
  * Total Orders

* Visual analysis:

  * Sales vs Profit by Category
  * Profit by Region
  * Monthly Sales Trend

* Interactive features:

  * Region slicer
  * Year slicer

## 🎯 Purpose

The dashboard is designed to simulate a real-world Business Analyst scenario, where stakeholders can explore performance metrics and identify areas for improvement.

## 🛠 Requirements

To view the dashboard:

* Install Power BI Desktop
* Open the `.pbix` file

## 📌 Notes

* The dataset used in this dashboard was cleaned and prepared in Excel before being imported into Power BI.
* A custom `order_month` column was created to ensure accurate time-series analysis.
